Yarn counts used by this calculator:
  • Cotton is the number of 840-yd hanks per pound.
  • Linen is the number of 300-yd hanks per pound.
  • Silk is the number of 840-yd hanks per pound.
  • Wool is the number of 560-yd hanks per pound.
There are many other yarn counts out there, but these are the ones with which weaving yarns seem to be typically labeled.
Note: Wool puts the ply count first, while other fibers put it second (after the /).
